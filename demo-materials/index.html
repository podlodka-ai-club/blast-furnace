Blast Furnace automated issue-to-PR workflow

A self-contained public demo of a private GitHub workflow: task intake, planning, development, quality checks, PR creation, human feedback, and PR rework.

Original Task

The source issue asked the agent to normalize category names by capitalizing the first letter when a user submits a category name.

Issue title

[Feature] Capitalize the first letter of Category name

A category like scientific books should be saved and displayed as Scientific books.

Acceptance criteria

  • New category creation applies first-letter capitalization.
  • Category rename submission follows the same rule.
  • Normalization happens on submission, not while typing.
  • Duplicate-name handling still runs after normalization.
  • Focused test coverage verifies the behavior.

Execution Pipeline

The orchestrator advanced the issue through deterministic stages, then accepted human PR feedback and routed the same PR through a rework pass.

  1. IntakePicked up a ready GitHub issue and created a run.
  2. Prepare RunCreated the working context for the target repository.
  3. AssessCompleted with a stub-safe assessment.
  4. PlanValidated a scoped implementation plan.
  5. DevelopExecuted the implementation with Codex.
  6. Quality GateRan the configured test command and passed.
  7. ReviewApproved the generated changes.
  8. Make PRCreated private PR #43.
  9. Tracker SyncMoved the task into review state.
  10. Human ReviewRequested a targeted rework fix.
  11. Rework DevelopUpdated the same PR.
  12. Final SyncMarked PR rework as finalized.

  • 2026-05-03 11:16ZPlan validated successfully and handed off to Develop.
  • 2026-05-03 11:21ZDevelop completed and Quality Gate passed.
  • 2026-05-03 11:22ZReview passed, Make PR created private PR #43, and tracker sync moved the task to review.
  • 2026-05-03 11:43ZHuman feedback requested rework; PR rework intake routed directly to Develop.
  • 2026-05-03 12:00ZRework passed Quality Gate and Review, updated the same PR, and finalized tracker state.
